# SCandMTest
Современная и безопасная форма обратной связи, реализованная на «чистом» стеке технологий без использования сторонних фреймворков и библиотек. Проект выполнен в рамках учебного/тестового задания с соблюдением архитектурного паттерна MVC и стандартов PSR-12. Проект сделан для "Скарлет и Монарх" в целях тестового задания :)
